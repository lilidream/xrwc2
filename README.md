# 仙人微彩模拟器 V2

## 功能
 - 统计玩家刮卡顺序、统计卡片号码等信息用于统计分析
 - 记录玩家分数与时间，显示图标并上传后台排名（上传需要加密验证，防止作弊）

以10张卡为一组，上传玩家开完一组卡的时间和总分数来排名，排名权重：总分数\*70%+用时(越短越好)\*30%
